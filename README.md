# Vision Transformer for Skin Disease Diagnosis

This project implements Vision Transformer (ViT-B/16) for dermatological image classification based on DermNet dataset.

## Features
- ViT (Transformer)
- Data preprocessing
- Training pipeline
- ROC curves
- Metrics (Accuracy, Precision, Recall, F1, AUC)

## Run
```bash
pip install -r requirements.txt
python main.py
